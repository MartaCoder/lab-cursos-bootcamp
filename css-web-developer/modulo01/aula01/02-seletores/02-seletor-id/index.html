Seletores CSS

Bem-vindo!

No CSS nós possuímos os seletores por ID

  • Seletor por tipo: Busca elementos por uma tag HTML específica
  • Seletor por ID(#): Busca elementos através do atributo "id"
  • Seletor por Class(.): Busca elementos através do atributo "class"
  • Seletor Universal(*): Seleciona todos os elemetos HTML
  • Seletor de atributo ([atrib]): Seleciona elementos que possuem em atributo especifíco em nosso documento HTML e conseguimos também buscar atributos com um valor especifíco
