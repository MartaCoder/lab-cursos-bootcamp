Cores

Existem várias formas de definirmos cores utilizando o CSS. Elas são:

  • Cores pré-definidas
  • Palavra-chave "currentcolor"
  • Hexadecimal
  • Hexadecimal com trasparência
  • RGB
  • RGBA
  • HLS
  • HLSA

HSL

hls(hue, sturation, lightness)
  • Hue (matriz): é o grau na roda de cores (de 0 a 360)
    • 0 ou 360
    • 120 é verde
    • 240 é azul
  • Saturation (saturação): valor percentual
    • 0% um tom de cinza
    • 100% cor total
  • Lightness (luminosidade): luminosidade de cor
    • 0% é preto
    • 100% é branco
