História dos computadores

Temos 5ª geração de computadores, ao longo das décadas que foram se desenvolvendo cada vez mais, acredito que estamos dando início a 5ª geração com o avanço da tecnologia principalmente a IA .
Como surgiu a internet? Bem na escola aprendemos sobre a guerra fria, onde o mundo foi dividido entre capitalismo e socialismo. Durante esses conflitos a rede, fundada nos EUA inicialmente chamada Arpanet, foi criada pela Darpa para facilitar o compartilhamento de informações e a comunicação entre diferentes sistemas de computador. A internet surgiu a partir de inovações para proteger as comunicações e informações. A popularização da internet ocorreu com o desenvolvimento da World Wide Web (WWW), criada por Tim Berners-Lee, e do navegador Mosaic.

O que são Clients?

O cliente envia uma solicitação: O cliente (por exemplo, um navegador) envia uma mensagem para o servidor, solicitando um determinado recurso (por exemplo, uma página web). O servidor processa a solicitação: O servidor recebe a solicitação, localiza o recurso solicitado e o prepara para ser enviado. O servidor envia a resposta: O servidor envia a resposta ao cliente, que pode ser uma página web, um arquivo, uma imagem, etc. O cliente exibe a resposta: O cliente recebe a resposta e a exibe para o usuário.
O cache é uma ferramenta fundamental para otimizar o desempenho dos computadores. Ao armazenar temporariamente dados frequentemente acessados, ele reduz o tempo de acesso aos dados e melhora a experiência do usuário.
Designer responsivo: É a pessoa que cria designs responsivos. Responsividade: É a característica de um design que se adapta a diferentes telas.
O que são aplicações web? Em poucas palavras, aplicações web são programas que você acessa e utiliza diretamente através de um navegador de internet, como o Google Chrome, Mozilla Firefox ou Safari. Diferentemente dos softwares tradicionais que você instala no seu computador, as aplicações web rodam em servidores remotos e são entregues aos seus dispositivos através da internet.

O que são Servers?

Imagine um servidor como um computador super poderoso e sempre ligado, pronto para atender a diversos pedidos ao mesmo tempo. Ele é o responsável por armazenar e fornecer informações para outros dispositivos conectados à internet, como seus computadores, smartphones e tablets. Esses outros dispositivos são chamados de clientes.

Como é uma estrutura básica de um HTML?

<!DOCTYPE html>: Declara o tipo de documento, informando ao navegador que se trata de um documento HTML5.
<html>: É a raiz de todo documento HTML, englobando todo o conteúdo. O atributo lang="pt-br" indica que o idioma principal do documento é o português do Brasil.
<head>: Contém informações sobre o documento, como o título, metadados e links para arquivos externos (como CSS).
<title>: Define o título da página, que aparece na aba do navegador.
<meta charset="UTF-8">: Especifica o conjunto de caracteres utilizado no documento, garantindo que caracteres especiais sejam exibidos corretamente.
<body>: Contém o conteúdo visível da página, ou seja, tudo o que o usuário verá na tela.
<h1> : Define um título de nível 1, o mais importante. Existem outros níveis de títulos (<h2>, <h3>, etc.).
<p>: Define um parágrafo. 

        
          <!DOCTYPE html>
          <html lang="pt-BR">
          <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Document</title >
          </head>
          <body>
            <h1>Titulo Maior</h1>
            <p>Paragrafo</p>
          </body>
          </html>

Entendendo a diferença entre listas ordenadas e não ordenadas

Listas Ordenadas (<ol>) Itens numerados: Cada item da lista possui um número, indicando sua posição na sequência. Ordem importante: A ordem dos itens é significativa e a numeração reflete essa ordem. Uso: Ideal para listas de passos, rankings, cronologias, etc.

  1. Primeiro lugar
  2. Segundo lugar
  3. Terceiro lugar

Listas Não Ordenadas (<ul>) Itens com marcadores: Cada item é precedido por um marcador, geralmente um ponto ou um círculo. Ordem não importante: A ordem dos itens não tem um significado específico. Uso: Ideal para listas de itens não relacionados por ordem, como itens de um menu, características de um produto, etc.
  • Arroz
  • Café
  • Açucar

Estrutura Ordenada 


        <ol>
        <li>Item 1</li>
        <li>Item 2</li>
        </ol>
Estrutura não Ordenada 

        <ul>
        <li>Item 1</li>
        <li>Item 2</li>
        </ul>
